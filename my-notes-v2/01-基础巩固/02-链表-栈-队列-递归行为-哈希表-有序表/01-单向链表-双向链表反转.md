## 一句话结论
链表反转的本质，就是把每个节点的指针方向整体翻过来。
单链表只需要改 `next`，双向链表则要同时交换 `next` 和 `pre` 的含义。

## 核心知识点
- 反转链表时，最怕的是“指针断掉”，所以一定要先保存下一个节点。
- 单链表反转的标准三指针是：`pre`、`head`、`next`。
- 双向链表反转时，每个节点的 `next` 和 `pre` 都要重新指向。
- 这类题的难点不是代码量，而是指针更新顺序。

## 单向链表为什么能这样反转
原链表是：

```text
1 -> 2 -> 3 -> null
```

反转后要变成：

```text
3 -> 2 -> 1 -> null
```

假设当前来到节点 `2`，我们想让它指向前一个节点 `1`，就必须先知道原来的下一个节点 `3` 在哪里。

所以顺序一定是：

1. 先记住 `next = head.next`
2. 再修改 `head.next = pre`
3. 再整体往后推进

如果不先保存 `next`，一旦 `head.next` 被改掉，后面的链表就找不回来了。

## 解题思路
### 1. 单向链表反转
准备三个变量：

- `pre`：当前节点反转后要指向的前一个节点
- `head`：当前正在处理的节点
- `next`：提前保存原链表中的下一个节点

循环过程中每次做 4 件事：

1. `next = head.next`
2. `head.next = pre`
3. `pre = head`
4. `head = next`

当 `head` 走到 `null` 时：

- `pre` 就停在新的头节点上

### 2. 双向链表反转
双向链表节点有两个方向：

- `next`
- `pre`

所以每处理一个节点时，要做两次改向：

1. 原来的 `next` 改成指向前一个节点 `pre`
2. 原来的 `pre` 改成指向原来的后一个节点 `next`

这样整个方向才算真正翻转完成。

## 典型例子
### 单向链表
假设链表为：

```text
1 -> 2 -> 3 -> null
```

初始状态：

- `pre = null`
- `head = 1`

第一轮：

- 保存 `next = 2`
- 改 `1.next = null`
- `pre = 1`
- `head = 2`

第二轮：

- 保存 `next = 3`
- 改 `2.next = 1`
- `pre = 2`
- `head = 3`

第三轮：

- 保存 `next = null`
- 改 `3.next = 2`
- `pre = 3`
- `head = null`

结束后返回 `pre`，得到：

```text
3 -> 2 -> 1 -> null
```

### 双向链表
原链表：

```text
null <- 1 <-> 2 <-> 3 -> null
```

反转后：

```text
null <- 3 <-> 2 <-> 1 -> null
```

以节点 `2` 为例：

- 原来 `2.pre = 1`
- 原来 `2.next = 3`

反转后要变成：

- `2.next = 1`
- `2.pre = 3`

也就是把两个方向一起翻过来。

## 复杂度
- 时间复杂度：`O(N)`
- 空间复杂度：`O(1)`

因为每个节点只访问一次，也只用了常数个额外指针变量。

## 易错点
- 没有先保存 `next`，导致后续节点丢失。
- 单链表反转结束后返回的是 `pre`，不是原来的 `head`。
- 双向链表只改了 `next`，忘了同步改 `pre`。
- 空链表和单节点链表要直接返回。

### 单链表迭代版

```java
ListNode pre = null;
ListNode next;
while (head != null) {
    next = head.next;
    head.next = pre;
    pre = head;
    head = next;
}
return pre;
```

### 单链表递归版

```java
if (head == null || head.next == null) {
    return head;
}
ListNode newHead = reverse(head.next);
head.next.next = head;
head.next = null;
return newHead;
```

### 双向链表迭代版

```java
DoubleListNode pre = null;
DoubleListNode next;
while (head != null) {
    next = head.next;
    head.next = pre;
    head.pre = next;
    pre = head;
    head = next;
}
return pre;
```

## 速记方法
- 单链表：先存后改，再推进。
- 双向链表：`next` 和 `pre` 一起翻。
- 返回值永远是“最后处理完的那个非空节点”，也就是 `pre`。
