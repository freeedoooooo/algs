# ADB Monitor 快速开始指南

## 🎯 5分钟快速上手

### 步骤1：准备工作

确保以下文件在同一目录：
```
adb-monitor/
├── monitor-gui.cmd       ← 双击这个文件启动
├── monitor-gui.ps1
├── monitor.config        ← 配置文件
├── start.ps1
├── stop.ps1
└── core/
    ├── check.ps1
    └── runner.ps1
```

### 步骤2：配置（可选）

编辑 `monitor.config` 文件，修改关键配置：

```ini
# 模拟器安装路径（根据实际位置修改）
ldplayer_path=D:\leidian\LDPlayer64

# 预期健康的设备数量
expected_healthy_devices=27

# 邮件通知（可选）
mail_enabled=false  # 如不需要邮件通知，设为 false
```

### 步骤3：启动

**双击 `monitor-gui.cmd`** 即可启动！

无需安装任何软件，Windows 自带 PowerShell 即可运行。

## 🔍 常见问题

### Q1: 双击后没反应？

**A:** 检查以下几点：
1. 是否所有文件都在同一目录
2. monitor.config 是否存在
3. PowerShell 版本是否在 3.0 以上

查看 PowerShell 版本：
```powershell
$PSVersionTable.PSVersion
```

### Q2: 提示"配置文件不存在"？

**A:** 确保 `monitor.config` 与 `monitor-gui.ps1` 在同一目录。

### Q3: 如何开机自启动？

**A:** 创建快捷方式到启动文件夹：
```
1. 右键 monitor-gui.cmd → 创建快捷方式
2. 按 Win+R，输入 shell:startup
3. 将快捷方式复制到打开的文件夹
```

### Q4: 如何批量部署到多台电脑？

**A:** 使用提供的批量部署工具：
```
1. 编辑 computers.txt，添加所有电脑名称
2. 以管理员身份运行 deploy.bat
3. 等待部署完成
```

详见：DEPLOYMENT.md

### Q5: 日志文件在哪里？

**A:** 默认在 `adb-monitor\log\` 目录下，文件名格式：`monitor-YYYYMMDD.log`

### Q6: 如何修改监控间隔？

**A:** 编辑 `monitor.config`：
```ini
schedule_interval_seconds=10  # 改为需要的秒数
```

## 💡 使用技巧

### 技巧1：快捷键

- `Ctrl+S` - 保存配置（在配置编辑器中）
- `F5` - 刷新日志（点击刷新日志按钮）

### 技巧2：查看历史日志

日志目录中会保留多天的日志文件：
```
log/
├── monitor-20260509.log
├── monitor-20260510.log
└── monitor-20260511.log  ← 今天的日志
```

### 技巧3：后台运行

关闭 GUI 窗口不会影响监控服务，服务会在后台继续运行。
