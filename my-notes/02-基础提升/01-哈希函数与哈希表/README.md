- 条目数量：7

## 本章概述
本章围绕哈希函数的核心性质展开，从基础原理出发，逐步延伸到哈希表实现、大数据分流、布隆过滤器、一致性哈希等工程场景，最后以岛问题收尾，引出并行化思路。

核心主线是：

```text
哈希函数的均匀性、离散性
-> 如何用它分桶、分流
-> 如何在此基础上设计高效数据结构
-> 如何在分布式场景下扩展应用
```
