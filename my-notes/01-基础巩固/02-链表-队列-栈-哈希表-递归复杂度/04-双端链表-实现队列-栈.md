## 一句话结论
双端链表只要支持“头尾插入”和“头尾删除”，就可以自然实现队列和栈。
队列强调先进先出，栈强调后进先出，本质区别只是选择从哪一端进、从哪一端出。

## 核心知识点
- 队列的规则是 FIFO：先进先出。
- 栈的规则是 LIFO：后进先出。
- 双端链表天然适合支持两端操作，所有核心操作都能做到 `O(1)`。
- 实现时需要维护 `head`、`tail` 和可选的 `size`。

## 为什么双端链表适合做队列和栈
上一题我们已经得到一个双端链表的能力：

- 从头加
- 从尾加
- 从头删
- 从尾删

而队列和栈其实只是对“出入规则”的限制：

### 队列
- 新元素通常从尾部进入
- 老元素从头部弹出

也就是：

```text
尾进，头出
```

### 栈
- 新元素从一端压入
- 也从同一端弹出

也就是：

```text
同端进，同端出
```

因此：

- 双端链表稍微限制一下使用方式，就能变成队列
- 再换一种限制方式，就能变成栈

## 解题思路
### 1. 用双端链表实现队列
队列遵循先进先出，所以可以规定：

- `offer` / `push`：从尾部加入
- `poll`：从头部弹出

为什么这样符合 FIFO：

- 先加入的元素更早靠近头部
- 后加入的元素排在更后面
- 每次总是从头部取元素，就保证了“先来先走”

### 2. 用双端链表实现栈
栈遵循后进先出，所以可以规定：

- `push`：从头部加入
- `pop`：从头部弹出

当然也可以全部放在尾部做：

- 尾部入栈
- 尾部出栈

本质一样，只要“从同一端进、从同一端出”即可。

### 3. 为什么时间复杂度都是 O(1)
因为双端链表已经直接保存了：

- `head`
- `tail`

所以不管是：

- 从头加 / 从头删
- 从尾加 / 从尾删

都不需要遍历整条链表。

## 典型例子
### 例1：队列
依次加入 `1、2、3`：

```text
head -> 1 <-> 2 <-> 3 <- tail
```

如果现在执行一次出队：

- 从头部取出 `1`

剩余：

```text
head -> 2 <-> 3 <- tail
```

再出队一次：

- 取出 `2`

这就是先进先出。

### 例2：栈
如果规定“都从头部操作”，依次压栈 `1、2、3`：

```text
head -> 3 <-> 2 <-> 1 <- tail
```

这时弹栈：

- 从头部弹出 `3`

再弹一次：

- 弹出 `2`

因此最后加入的元素最先出去，符合后进先出。

## 边界情况
- 空结构上执行 `poll` / `pop`：通常返回 `null` 或抛异常，取决于设计
- 只有一个节点时删除：删完后 `head` 和 `tail` 都要变成 `null`
- 删除头节点后，新头节点的 `pre` 要置空
- 删除尾节点后，新尾节点的 `next` 要置空

## 复杂度
- 入队 / 出队：`O(1)`
- 入栈 / 出栈：`O(1)`
- 空间复杂度：`O(N)`

这里的 `O(N)` 是结构存储 `N` 个节点所需空间，不是额外辅助空间。

## 易错点
- 只更新了 `head`，忘了同步更新 `tail`。
- 删除最后一个节点时，没有把两个边界都清空。
- 删除头节点后，忘了把新头的 `pre` 设为 `null`。
- 删除尾节点后，忘了把新尾的 `next` 设为 `null`。
- 实现栈时，如果一会儿从头操作、一会儿从尾操作，就会把规则搞乱。

### 双端链表版队列

```java
class MyDequeQueue {
    DoubleListNode head;
    DoubleListNode tail;

    void offer(int value) {
        DoubleListNode node = new DoubleListNode(value);
        if (tail == null) {
            head = tail = node;
            return;
        }
        tail.next = node;
        node.pre = tail;
        tail = node;
    }

    Integer poll() {
        if (head == null) {
            return null;
        }
        int ans = head.val;
        if (head == tail) {
            head = tail = null;
        } else {
            head = head.next;
            head.pre = null;
        }
        return ans;
    }
}
```

### 双端链表版栈

```java
class MyDequeStack {
    DoubleListNode head;

    void push(int value) {
        DoubleListNode node = new DoubleListNode(value);
        if (head != null) {
            node.next = head;
            head.pre = node;
        }
        head = node;
    }

    Integer pop() {
        if (head == null) {
            return null;
        }
        int ans = head.val;
        head = head.next;
        if (head != null) {
            head.pre = null;
        }
        return ans;
    }
}
```

## 速记方法
- 队列：尾进头出。
- 栈：同端进，同端出。
- 双端链表：谁要从哪端操作，就直接维护那一端的边界指针。
