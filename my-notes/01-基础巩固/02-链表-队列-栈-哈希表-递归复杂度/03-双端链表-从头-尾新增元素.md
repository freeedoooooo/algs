## 一句话结论
双端链表的价值，在于可以在头部和尾部都做到 `O(1)` 插入。
实现时只要维护好 `head`、`tail` 两个边界指针，并在插入时同时修正新节点与相邻节点的双向关系即可。

## 核心知识点
- 双端链表节点有两个方向：`pre` 和 `next`。
- 结构层面通常至少维护两个指针：`head` 和 `tail`。
- 头插和尾插都属于常数时间操作。
- 最容易出错的是空链表插入，以及忘记同步更新相邻节点的反向指针。

## 什么是双端链表
双端链表可以理解为“双向链表 + 头尾两个入口”。

也就是说：

- 你知道链表头在哪里
- 也知道链表尾在哪里
- 每个节点既知道前驱，也知道后继

节点结构通常类似这样：

```java
class DoubleListNode {
    int val;
    DoubleListNode pre;
    DoubleListNode next;
}
```

仓库里的节点定义就在：

- [DoubleListNode.java](../../../solutions-java/util/collection/DoubleListNode.java)

## 为什么头插、尾插都能做到 O(1)
因为双端链表已经直接保存了：

- 头节点 `head`
- 尾节点 `tail`

所以：

- 从头新增时，不需要遍历去找头，因为头本来就知道
- 从尾新增时，也不需要遍历去找尾，因为尾也本来就知道

这和单链表很不一样：

- 单链表如果只有头指针，尾插往往需要先走到最后一个节点

## 解题思路
### 1. 头插
准备一个新节点 `node`，把它插到链表最前面。

#### 空链表情况
如果当前链表为空：

- `head = node`
- `tail = node`

因为这是链表中的唯一节点。

#### 非空链表情况
如果当前链表不为空，原来的头节点记为 `oldHead`：

1. `node.next = oldHead`
2. `oldHead.pre = node`
3. `head = node`

这样新节点就成为新的头。

### 2. 尾插
准备一个新节点 `node`，把它插到链表最后面。

#### 空链表情况
仍然一样：

- `head = node`
- `tail = node`

#### 非空链表情况
如果原来的尾节点记为 `oldTail`：

1. `oldTail.next = node`
2. `node.pre = oldTail`
3. `tail = node`

这样新节点就成为新的尾。

## 典型例子
### 例1：从头插入
原链表：

```text
null <- 2 <-> 3 -> null
```

现在头插一个值为 `1` 的节点。

连接过程：

1. `1.next = 2`
2. `2.pre = 1`
3. `head = 1`

结果：

```text
null <- 1 <-> 2 <-> 3 -> null
```

### 例2：从尾插入
原链表：

```text
null <- 1 <-> 2 -> null
```

现在尾插一个值为 `3` 的节点。

连接过程：

1. `2.next = 3`
2. `3.pre = 2`
3. `tail = 3`

结果：

```text
null <- 1 <-> 2 <-> 3 -> null
```

## 边界情况
- 空链表插入：`head` 和 `tail` 都要同时指向新节点
- 只有一个节点时继续头插或尾插：逻辑和普通非空链表一致
- 新节点刚创建时，它的 `pre` 和 `next` 默认应为 `null`
- 如果忘了更新 `head` 或 `tail`，链表外部入口就会失效

## 复杂度
- 头插时间复杂度：`O(1)`
- 尾插时间复杂度：`O(1)`
- 空间复杂度：`O(1)` 额外空间

## 易错点
- 空链表插入时，只更新了 `head`，忘了同步更新 `tail`。
- 头插时忘了让原头节点的 `pre` 指向新节点。
- 尾插时忘了让原尾节点的 `next` 指向新节点。
- 链表只有一个节点时，把边界判断写复杂反而更容易出错。

## 代码 / 伪代码
仓库里虽然没有单独封装“头插/尾插”的类，但节点结构已经准备好：

- [DoubleListNode.java](../../../solutions-java/util/collection/DoubleListNode.java)

头插可以写成：

```java
public void addFromHead(int value) {
    DoubleListNode node = new DoubleListNode(value);
    if (head == null) {
        head = node;
        tail = node;
        return;
    }
    node.next = head;
    head.pre = node;
    head = node;
}
```

尾插可以写成：

```java
public void addFromTail(int value) {
    DoubleListNode node = new DoubleListNode(value);
    if (tail == null) {
        head = node;
        tail = node;
        return;
    }
    tail.next = node;
    node.pre = tail;
    tail = node;
}
```

- 队列

## 速记方法
- 头插：新节点连旧头，`head` 向前挪。
- 尾插：旧尾连新节点，`tail` 向后挪。
- 空链表时，`head` 和 `tail` 一起指向新节点。
