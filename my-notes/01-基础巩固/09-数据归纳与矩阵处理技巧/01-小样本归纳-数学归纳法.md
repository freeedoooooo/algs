## 一句话结论
这类题的核心，不是“拍脑袋猜答案”，而是：

```text
先写出小范围一定正确的暴力解
再把小样本答案系统列出来
最后从结果中观察稳定规律，并压缩成公式或 O(1) 判断
```

这类题本质上是：

```text
小范围枚举 + 规律观察 + 数学归纳
```

## 先记结论
第一次看这篇，建议先记住下面 4 句话：

```text
1. 这类归纳题通常参数很少，返回值也很简单
2. 第一步不是找公式，而是先写暴力正确解
3. 重点看奇偶、取模、周期、临界点、幂次点
4. 最终目标不是保留样本表，而是把规律压成快解
```

最常见的节奏就是：

```text
暴力算小样本
-> 列出结果表
-> 观察是否有稳定模式
-> 用数学归纳或分类讨论写成 O(1)
```

## 这篇到底在讲什么
这篇不是某一道固定题，而是在讲一种“解题套路”：

```text
有些题你正面推公式会很难
但先暴力列出 20 个、50 个、100 个样本
反而很容易看出规律
```

所以它训练的不是某个具体技巧，而是一种思考顺序：

```text
先别急着推结论
先把现象完整地列出来
```

## 什么时候该想到小样本归纳
遇到下面这种题，就要优先怀疑它能不能“先枚举小样本，再做规律归纳”：

- 只有一个主要参数，或者参数很少
- 返回值类型简单，比如布尔值、赢家、最小值、步数
- 小范围暴力解能写出来，而且能保证正确
- 题目看起来像有周期、奇偶、分类讨论的味道

这种题最怕的不是算慢，而是：

```text
一上来就想推公式
结果越推越乱
```

## 这类题的统一流程
可以把这类题固定成 4 步。

### 第 1 步：先写一个绝对正确的暴力解
这一步不要在乎复杂度，只在乎：

```text
小范围输入下，答案绝对正确
```

因为如果暴力版都写错了，后面观察出来的“规律”一定也是错的。

### 第 2 步：把小样本完整列出来
比如直接打出：

```text
0 ~ 50
0 ~ 100
```

对应的结果表。

这里不是只看几个点，而是要尽量把一整段现象铺开。

### 第 3 步：重点观察 5 类规律
最值得先看的通常是：

- 奇偶性
- 对某个数取模后的分类
- 是否有周期性
- 是否从某个临界值开始变稳定
- 是否出现特殊点，比如 `1, 2, 4, 8, 16`

### 第 4 步：把规律压成快解
最后真正要交的，不是那张结果表，而是压缩后的结论。

常见形式有：

- `if-else` 小范围分类
- `N % k`
- 位运算
- 从某个阈值开始的通项公式

## 图解
这类题可以统一理解成下面这条流水线：

```text
题目
-> 写暴力
-> 列小样本结果表
-> 观察规律
-> 总结成快解
```

你真正要练熟的，不是某一张表，而是这条流程。

## 例题 1：苹果装袋
### 题意
商店只有两种袋子：

- 每袋装 `6` 个苹果
- 每袋装 `8` 个苹果

每种袋子数量无限，并且每个袋子都必须装满。  
给定正整数 `N`，返回装下 `N` 个苹果的最少袋数；如果刚好装不出来，返回 `-1`。

### 先列样本

```text
N   结果
1   -1
6    1
8    1
10  -1
12   2
14   2
16   2
18   3
20   3
22   3
24   3
26   4
28   4
30   4
```

### 再观察规律
从这张表里最容易先看到 3 件事：

1. 奇数一定不行  
   因为 `6` 和 `8` 都是偶数，偶数加偶数永远凑不出奇数。

2. 小于 `18` 的偶数里，不是每个都能装  
   只有：

```text
6, 8, 12, 14, 16
```

能成立。

3. 从 `18` 开始，规律稳定了  
   每多 `8` 个苹果，最少袋数就多 `1`。

### 最后压成结论

```text
奇数：直接 -1
小于 18：单独分类讨论
大于等于 18：按 (N - 18) / 8 + 3 计算
```

### 这题最该学到什么
这题最重要的不是苹果，而是：

```text
很多题前面会有一小段“乱区”
但从某个临界点开始，规律会稳定
```

## 例题 2：牛羊吃草
### 题意
给定正整数 `N`，表示仓库里有 `N` 份草。

- 牛先吃，羊后吃
- 双方轮流行动
- 每次只能吃 `1, 4, 16, 64...`，也就是 `4` 的某次方
- 谁先把草吃完谁赢
- 双方都绝顶聪明，都会采取最优策略

问最后谁会赢。

### 先列样本

```text
N   赢家
0   羊
1   牛
2   羊
3   牛
4   牛
5   羊
6   牛
7   羊
8   牛
9   牛
10  羊
```

### 再观察规律
一开始可能看不出来，但多打几项后会发现：

```text
0, 2, 5, 7, 10, 12 ...
```

这些后手赢的位置呈现稳定周期。  
继续归纳后可以压成：

```text
N % 5 == 0 或 2 -> 后手赢
其他情况         -> 先手赢
```

因为题目里后手是羊，所以也可以写成：

```text
N % 5 == 0 或 2 -> 羊赢
其他情况         -> 牛赢
```

### 最后压成结论
这一题的最终快解不是搜索，而是：

```text
看 N % 5
```

### 这题最该学到什么
这题最关键的训练点是：

```text
博弈题的小样本表里
经常藏着“取模周期”
```

## 例题 3：连续正数和
### 题意
给定正整数 `N`，判断它是否能表示成：

- 项数大于 `1`
- 若干个连续正整数的和

例如：

- `5 = 2 + 3`
- `12 = 3 + 4 + 5`

但：

- `1` 不行，因为项数不够
- `2 = 1 + 1` 也不行，因为不是连续正整数

### 先列样本

```text
N   是否可以表示
1   否
2   否
3   是   1+2
4   否
5   是   2+3
6   是   1+2+3
7   是   3+4
8   否
9   是   4+5
10  是   1+2+3+4
```

### 再观察规律
继续往下打，会发现不能表示的数很特别：

```text
1, 2, 4, 8, 16 ...
```

也就是：

```text
2 的幂
```

### 最后压成结论
所以答案可以直接压成：

```text
N 只要不是 2 的幂，就一定可以表示
N 如果是 2 的幂，就不可以表示
```

进一步用位运算写成：

```text
(n & (n - 1)) == 0  <=>  n 是 2 的幂
```

### 这题最该学到什么
这题提醒你：

```text
做样本归纳时不仅要看奇偶和取模
还要留意“幂次点”这种特殊序列
```

## 三个例题放在一起看
这三题其实分别代表了小样本归纳里最常见的三种规律来源：

| 例题 | 最终规律 | 你该重点观察什么 |
| --- | --- | --- |
| 苹果装袋 | 临界点后稳定递推 | 奇偶 + 临界点 |
| 牛羊吃草 | `N % 5` 周期 | 小样本周期 / 取模 |
| 连续正数和 | `2` 的幂是例外 | 特殊序列 / 位运算 |

把它们压成一句话就是：

```text
列样本不是为了记题
而是为了训练你看到“规律属于哪一类”
```

## 复杂度
- 暴力枚举阶段：取决于暴力解本身，通常比最终解更慢
- 规律版求解：常见是 `O(1)`
- 额外空间复杂度：常见是 `O(1)`

## 易错点
- 列样本不是“猜规律”，必须先有正确暴力解。
- 小样本一定要列得足够长，否则很容易把假规律当真规律。
- 看到规律后，最好再拿更多样本回验一次。
- 苹果装袋题里，“偶数”只是必要条件，不是充分条件。
- 连续正数和这题，真正的判断标准不是奇偶，而是不是 `2` 的幂。

## 代码 / 伪代码
仓库里目前没有这一题对应的现成实现文件，这里保留课程标准伪代码。

### 1. 苹果装袋

```java
int minBags(int n) {
    if ((n & 1) != 0) {
        return -1;
    }
    if (n < 18) {
        if (n == 0) return 0;
        if (n == 6 || n == 8) return 1;
        if (n == 12 || n == 14 || n == 16) return 2;
        return -1;
    }
    return (n - 18) / 8 + 3;
}
```

### 2. 牛羊吃草

```java
String winner(int n) {
    return (n % 5 == 0 || n % 5 == 2) ? "羊" : "牛";
}
```

### 3. 连续正数和

```java
boolean canSplit(int n) {
    return (n & (n - 1)) != 0;
}
```

其中：

```text
n 是 2 的幂 <=> (n & (n - 1)) == 0
```

## 记忆点
- 小样本归纳的核心是：先暴力，后观察，再压缩。
- 最常见规律来源：奇偶、取模、临界点、幂次点。
- 苹果装袋看“奇偶 + 临界点”。
- 牛羊吃草看“周期 / 取模”。
- 连续正数和看“是不是 2 的幂”。
