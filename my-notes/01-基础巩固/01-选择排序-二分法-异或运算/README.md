## 章节导读
这一章看起来只有 3 个小题，但其实是整套算法学习的三个起点：

- 排序入门：选择排序
- 查找与剪枝思维：二分法
- 位运算基础：异或

如果把知识结构压缩成一句话，就是：

```text
会写最基础的排序
会理解“排除一半”的判断逻辑
会使用最常见的位运算性质
```

## 本章结论
- 选择排序帮助建立“数组分区处理”的最基础感觉。
- 二分法的核心不是“数组有序”四个字，而是“每一步都能排除一半不可能答案”。
- 异或最重要的不是计算，而是利用抵消、交换、分组等性质解决题目。
- 这三题虽然简单，但后面很多题都会反复调用这里的基本认知。

## 章节笔记还原
### 1. 选择排序是在建立“流程感”
选择排序本身并不是最强排序算法，但它非常适合拿来训练两个基础观念：

- 数组可以分成“已经处理完”和“还没处理”的两部分
- 每一轮循环都应该有一个非常明确的目标

在选择排序里，这个目标就是：

```text
第 i 轮
在 i...n-1 中选最小值
放到 i 位置
```

这一点很重要，因为后面的插入排序、归并排序、快速排序，本质上也都在做“分区 + 目标推进”。

### 2. 二分法是在建立“排他性判断”
这一章的二分题不是普通“找某个数”，而是“局部最小”。

它想强调的是：

- 二分并不要求整个数组全局有序
- 它真正要求的是：你看完 `mid` 后，能够明确砍掉一边

所以二分的关键词不该只记“有序”，而应该记：

```text
单调性
边界性
排他性
```

这会直接影响后面你能不能看懂：

- 最左满足条件位置
- 最右满足条件位置
- 峰值 / 谷值
- 答案区间二分

### 3. 异或是在建立“位运算直觉”
异或这题通常是很多人第一次真正把位运算当工具来用。

最值得记住的是几条性质：

- 相同为 `0`，不同为 `1`
- `a ^ 0 = a`
- `a ^ a = 0`
- 异或满足交换律、结合律

所以它特别适合处理：

- 一个数出现奇数次、其他数出现偶数次
- 两个数交换
- 按某一位把数据分成两组

### 4. 这三题之间的关系
别看它们题型不同，但底层训练的是同一种能力：

- 选择排序练的是“把大问题拆成循环过程”
- 二分法练的是“用性质缩小搜索范围”
- 异或练的是“利用运算规则做简化”

它们共同构成了后面做题时最底层的三种动作：

```text
遍历
剪枝
化简
```
