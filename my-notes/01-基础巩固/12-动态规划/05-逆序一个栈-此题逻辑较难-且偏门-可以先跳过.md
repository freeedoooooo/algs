## 一句话结论
这题要求只能用递归函数，不能申请额外数据结构，把一个栈整体逆序。难点在于先设计一个辅助递归函数，把“栈底元素”单独拿出来。
一旦能递归地拿到底元素，剩下就是“先逆序剩余部分，再把底元素压回栈顶”。

## 核心知识点
- 不能借助额外栈或数组。
- 关键辅助函数：返回并移除栈底元素。
- 主函数先递归逆序剩余部分，再把原栈底压回去。
- 这题本质是在练“递归过程本身就是隐式栈”。

## 图片转写 / 题意还原
原图内容为：

- 给你一个栈，请你逆序这个栈
- 不能申请额外的数据结构
- 只能使用递归函数
- 问如何实现

## 图解
栈顶到栈底：

```text
1
2
3
4
```

目标逆序后变成：

```text
4
3
2
1
```

关键步骤：

1. 先把栈底 `4` 递归拿出来
2. 对 `[1,2,3]` 继续逆序
3. 再把 `4` 压回栈顶

## 解题思路
### 1. 设计 `getBottom`
功能：

- 弹出栈顶
- 如果弹出后栈空，说明刚刚拿到的是原栈底，直接返回
- 否则递归继续找底
- 回溯时把之前弹出的元素再压回去

### 2. 主递归 `reverse`
- 栈空时结束
- 先拿到底元素 `bottom`
- 递归逆序剩余部分
- 最后把 `bottom` 压回去

## 复杂度
- 时间复杂度：`O(N^2)`
- 空间复杂度：`O(N)`，来自递归栈

## 典型例子
对 `[1,2,3]`：

- `getBottom` 返回 `3`
- 递归逆序 `[1,2]`
- 再把 `3` 压回

最终得到 `[3,2,1]`。

## 易错点
- “拿栈底”时，回溯阶段要把中间元素压回去。
- 主函数和辅助函数职责不同，不要混写。
- 这题不允许显式申请额外栈。

## 代码 / 伪代码
课程标准伪代码：

```java
int getBottom(Stack<Integer> stack) {
    int top = stack.pop();
    if (stack.isEmpty()) {
        return top;
    }
    int last = getBottom(stack);
    stack.push(top);
    return last;
}

void reverse(Stack<Integer> stack) {
    if (stack.isEmpty()) {
        return;
    }
    int bottom = getBottom(stack);
    reverse(stack);
    stack.push(bottom);
}
```

## 记忆点
- 先设计“拿栈底”。
- 栈底拿出来，剩余部分递归逆序。
- 回溯阶段把中间元素放回去。
- 这题靠的是系统栈。
