## 一句话结论
纸牌博弈的递归状态是区间 `[L, R]`，所以最自然的 DP 就是两张二维表：先手表 `fdp` 和后手表 `sdp`。

## 解题思路
- `fdp[L][R]`：先手在 `[L,R]` 上能拿到的最好分数
- `sdp[L][R]`：后手在 `[L,R]` 上能拿到的最好分数
- 对角线是 base case
- 按区间长度从小到大填

## 代码 / 伪代码
```java
for (int i = 0; i < N; i++) {
    fdp[i][i] = arr[i];
    sdp[i][i] = 0;
}
for (int len = 2; len <= N; len++) {
    for (int L = 0; L + len - 1 < N; L++) {
        int R = L + len - 1;
        fdp[L][R] = Math.max(arr[L] + sdp[L + 1][R],
                             arr[R] + sdp[L][R - 1]);
        sdp[L][R] = Math.min(fdp[L + 1][R],
                             fdp[L][R - 1]);
    }
}
return Math.max(fdp[0][N - 1], sdp[0][N - 1]);
```

## 记忆点
- 范围模型常开二维表。
- 先手 / 后手通常是两张表。
- 先填对角线，再按斜线扩。
