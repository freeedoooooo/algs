## 一句话结论
这题本质上就是前面“二叉树最大宽度”的直接问法落地版，答案就是整棵树所有层节点数中的最大值。
最自然的做法仍然是按层遍历，每处理完一层，就更新一次这一层的节点数量最大值。

## 核心知识点
- “最宽的层有多少个节点” = “二叉树最大宽度”。
- 基础解法仍然是 BFS，也就是队列按层遍历。
- 每层开始时的 `queue.size()`，就是当前层节点数。
- 也可以用 `Map` 或“当前层结束节点”来做分层。
- 这一题更像是在考你能不能把按层遍历真正落成返回值。

## 图片转写 / 题意还原
原始笔记里只有题目名，没有额外说明。

但这题和上一题其实是同一个问题的另一种表述：

```text
上一题问：二叉树最大宽度是多少
这一题问：最宽的那一层有多少个节点
```

所以本质完全一致。

例如：

```text
        1
      /   \
     2     3
    / \     \
   4   5     6
```

每层节点数分别是：

- 第 1 层：1
- 第 2 层：2
- 第 3 层：3

因此最宽层的节点数就是：

```text
3
```

## 图解
还是看这棵树：

```text
        1
      /   \
     2     3
    / \     \
   4   5     6
```

按层遍历时：

```text
第 1 层队列：[1]         -> 本层节点数 1
第 2 层队列：[2, 3]      -> 本层节点数 2
第 3 层队列：[4, 5, 6]   -> 本层节点数 3
```

所以：

```text
max = max(1, 2, 3) = 3
```

## 解题思路
### 1. 最直接写法：用 `queue.size()`
这是最简洁也最容易写对的方法。

做法：

1. 根节点入队
2. 进入循环时，先取当前 `queue.size()`
3. 这个值就是“当前层节点数”
4. 用它更新最大值
5. 再把这一层节点全部弹出，并把它们的孩子加入队列

伪代码：

```java
queue.offer(head);
int max = 0;
while (!queue.isEmpty()) {
    int levelSize = queue.size();
    max = Math.max(max, levelSize);
    for (int i = 0; i < levelSize; i++) {
        Node cur = queue.poll();
        if (cur.left != null) {
            queue.offer(cur.left);
        }
        if (cur.right != null) {
            queue.offer(cur.right);
        }
    }
}
```

### 2. 也可以写成 `Map` 版
如果想更显式地知道“每个节点属于哪一层”，也可以：

- 用 `Map<Node, Integer>` 记录层号
- 按层号统计当前层节点数
- 层号变化时更新答案

这种写法更适合理解分层，但代码更啰嗦。

### 3. 也可以写成“不用 `Map` 的层边界版”
还可以用：

- `curEnd`：当前层最后一个节点
- `nextEnd`：下一层最后一个节点

来判断什么时候一层结束。

这个版本本质上和上一题完全一样，只是最后返回的是：

```text
最大层节点数
```

## 典型例子
用这棵树：

```text
    1
   / \
  2   3
 /
4
```

每层节点数：

```text
第 1 层：1
第 2 层：2
第 3 层：1
```

所以答案：

```text
2
```

再看一棵更偏右的树：

```text
1
 \
  2
   \
    3
```

每层都只有 1 个节点，所以答案：

```text
1
```

## 复杂度
- 时间复杂度：`O(N)`
- 空间复杂度：`O(W)`，最坏为 `O(N)`

其中：

- `N` 是节点数
- `W` 是某一层的最大节点数

## 易错点
- 这题和上一题其实是同一道模型题，不要当成全新套路。
- 不要把“带空位的宽度”混进来，这里只算真实存在的节点数。
- 如果用 `queue.size()`，要先保存下来再循环，不能在遍历过程中动态读变化后的大小。
- 空树要返回 `0`。

## 记忆点
- 这题就是“二叉树最大宽度”的另一种问法。
- 解法核心还是按层遍历。
- `queue.size()` 就是当前层节点数。
- 每层统计一次，取最大值即可。
