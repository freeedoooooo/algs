## 一句话结论
先序、中序、后序遍历并不是 3 套完全独立的东西，它们本质上都来自同一个递归框架。
唯一的区别只是：在递归序的“第一次到达、第二次到达、第三次到达”这 3 个时机里，你把打印动作放在哪个位置。

## 核心知识点
- 先序遍历：第一次到达节点时处理。
- 中序遍历：第二次到达节点时处理。
- 后序遍历：第三次到达节点时处理。
- 理解这一点之后，三序遍历会从“死记模板”变成“自然推出来”。

## 图片转写 / 题意还原
原始笔记虽然只有题目名称，但这题其实就是上一题“理解递归序”的直接应用：

- 递归序是底层统一框架
- 先序、中序、后序只是加工结果

所以这题重点不是多写 3 个函数，而是理解：

```text
同一套递归框架
在不同位置打印
就得到不同遍历顺序
```

## 图解
假设二叉树：

```text
        1
      /   \
     2     3
    / \   / \
   4   5 6   7
```

先看最基础的递归框架：

```java
void f(Node head) {
    if (head == null) {
        return;
    }
    // 位置1
    f(head.left);
    // 位置2
    f(head.right);
    // 位置3
}
```

对于节点 `1` 来说：

```text
第一次到达 1：刚进入函数
第二次到达 1：左子树处理完返回
第三次到达 1：右子树处理完返回
```

如果在：

```text
位置1打印 -> 先序
位置2打印 -> 中序
位置3打印 -> 后序
```

那么这棵树就会得到：

```text
先序：1 2 4 5 3 6 7
中序：4 2 5 1 6 3 7
后序：4 5 2 6 7 3 1
```

可以把它记成一句话：

```text
先序：根在前
中序：根在中
后序：根在后
```

## 解题思路
### 1. 先写统一递归框架
不管是哪种遍历，骨架都一样：

```java
if (head == null) return;
递归左树
递归右树
```

真正变化的只是“处理当前节点”的时机。

### 2. 先序遍历
如果在递归左树之前先处理当前节点：

```java
print(head);
preOrder(head.left);
preOrder(head.right);
```

那么顺序就是：

```text
根 -> 左 -> 右
```

### 3. 中序遍历
如果在左树处理完之后、右树处理之前处理当前节点：

```java
inOrder(head.left);
print(head);
inOrder(head.right);
```

那么顺序就是：

```text
左 -> 根 -> 右
```

### 4. 后序遍历
如果左右子树都处理完之后再处理当前节点：

```java
posOrder(head.left);
posOrder(head.right);
print(head);
```

那么顺序就是：

```text
左 -> 右 -> 根
```

## 典型例子
还是用这棵树：

```text
    1
   / \
  2   3
```

### 先序

```text
1 2 3
```

因为每次一进节点就打印。

### 中序

```text
2 1 3
```

因为总是先把左边走完，再打印根。

### 后序

```text
2 3 1
```

因为左右都处理完了，最后才轮到根。

## 复杂度
- 时间复杂度：`O(N)`
- 空间复杂度：`O(H)`

其中：

- `N` 是节点数
- `H` 是树高

空间复杂度来自递归栈深度。

## 易错点
- 先序 / 中序 / 后序的区别，不在“递归方向”，而在“打印时机”。
- 不要把“第一次到达、第二次到达、第三次到达”理解成 3 次递归调用，它们是同一次函数生命周期中的 3 个时机。
- 写代码时三种遍历的 `if (head == null) return;` 都一样，不同的是打印语句的位置。
- 这题如果没和上一题“递归序”连起来看，很容易退化成死记模板。

对应三种写法分别是：

```java
// 先序
System.out.print(head.val + " ");
preOrder(head.left);
preOrder(head.right);
```

```java
// 中序
inOrder(head.left);
System.out.print(head.val + " ");
inOrder(head.right);
```

```java
// 后序
posOrder(head.left);
posOrder(head.right);
System.out.print(head.val + " ");
```

## 记忆点
- 先序：第一次到达就处理。
- 中序：左边回来后处理。
- 后序：左右都处理完再处理。
- 三序遍历只是递归序的 3 个落点选择题。
