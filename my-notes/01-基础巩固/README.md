## 模块导读
`基础巩固` 这一大类不是单纯按题号堆起来的，而是按“能力成长路径”排的。

如果把这 16 章压缩成一条学习路线，可以这样理解：

```text
数组 / 链表 / 栈队列
-> 排序与堆
-> 前缀树与非比较排序
-> 链表面试高频题
-> 二叉树遍历与递归套路
-> 打表、矩阵、并查集、图
-> 暴力递归
-> 动态规划
```

也就是说，这一部分真正训练的是：

- 数据结构基本操作能力
- 常见算法模板能力
- 递归与动态规划建模能力
