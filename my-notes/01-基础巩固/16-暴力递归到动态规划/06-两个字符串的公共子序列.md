## 一句话结论
最长公共子序列是“多样本位置全对应模型”的代表题。状态通常是两个位置参数，表示分别来到两个字符串的哪里。

## 核心知识点
- 经典目标是求 LCS 长度。
- 状态常用 `i, j`，表示考虑 `str1[0..i]` 与 `str2[0..j]`。
- 转移通常围绕“最后一个字符要不要配对”展开。

## 解题思路
经典转移：

- 如果 `str1[i] == str2[j]`，可以来自 `dp[i-1][j-1] + 1`
- 否则看：
  - `dp[i-1][j]`
  - `dp[i][j-1]`

完整写法也常先枚举：

- 不要 `str1[i]`
- 不要 `str2[j]`
- 两者都要且相等

## 代码 / 伪代码
```java
for (int i = 1; i < N; i++) {
    for (int j = 1; j < M; j++) {
        dp[i][j] = Math.max(dp[i - 1][j], dp[i][j - 1]);
        if (str1[i] == str2[j]) {
            dp[i][j] = Math.max(dp[i][j], dp[i - 1][j - 1] + 1);
        }
    }
}
```

## 记忆点
- LCS 是双样本位置模型。
- 状态通常是两个下标。
- 相等时才考虑斜对角加一。
