## 一句话结论
堆本质上不是“链式树结构”，而是“用数组实现的完全二叉树”。
只要满足完全二叉树的形状，再额外满足堆序性质，就能分别得到大根堆或小根堆。

## 图片转写
原图内容可以整理成下面这份文字版要点：

1. 堆结构就是用数组实现的完全二叉树结构。
2. 如果每棵子树的最大值都在顶部，就是大根堆。
3. 如果每棵子树的最小值都在顶部，就是小根堆。
4. 堆结构的核心操作是 `heapInsert` 与 `heapify`。
5. 堆结构支持元素的增加和减少。
6. 优先级队列结构，本质上就是堆结构。

## 核心知识点
- 堆先是完全二叉树，后才谈大根堆/小根堆。
- 完全二叉树强调“从上到下、从左到右依次变满”。
- 大根堆要求每个节点都不小于自己的孩子。
- 小根堆要求每个节点都不大于自己的孩子。
- 堆最重要的两个调整动作是向上调整 `heapInsert` 和向下调整 `heapify`。

## 图解
数组：

```text
[9, 7, 8, 3, 5, 6]
```

可以画成完全二叉树：

```text
        9
      /   \
     7     8
    / \   /
   3   5 6
```

### 大根堆

```text
父节点 >= 子节点
```

所以上图就是大根堆。

### 小根堆
如果变成：

```text
        1
      /   \
     3     2
    / \   /
   7   5 6
```

那就是小根堆，因为每个父节点都最小。

## 什么是完全二叉树
完全二叉树不是“每层都满”，而是：

- 除最后一层外，上面的层都满
- 最后一层从左到右连续排布

这也是为什么堆非常适合用数组实现。

如果下标从 `0` 开始，那么：

- 父节点下标：`(i - 1) / 2`
- 左孩子下标：`2 * i + 1`
- 右孩子下标：`2 * i + 2`

因为结构足够规整，所以不需要真的建树节点和指针。

## 什么是大根堆 / 小根堆
### 大根堆
对任意一个节点，都满足：

```text
父节点 >= 左孩子
父节点 >= 右孩子
```

所以整棵堆的最大值一定在堆顶。

### 小根堆
对任意一个节点，都满足：

```text
父节点 <= 左孩子
父节点 <= 右孩子
```

所以整棵堆的最小值一定在堆顶。

## 为什么堆顶能代表全局最值
因为堆要求“每个局部父子关系”都满足大小约束。

以大根堆为例：

- 根节点不小于第一层孩子
- 第一层每个节点又不小于自己的孩子
- 这种关系会一路传递下去

所以根节点自然就是全局最大值。

## 解题思路
### 1. 先理解“形状”
堆的第一层限制不是大小，而是形状：

- 必须是完全二叉树
- 数组从左到右连续放元素

### 2. 再理解“堆序”
在完全二叉树的基础上，再加一个规则：

- 大根堆：父 >= 子
- 小根堆：父 <= 子

### 3. 建堆靠两个核心操作
#### `heapInsert`
当新元素加到数组末尾时，它可能破坏了堆序。

做法：

- 不断和父节点比较
- 如果不满足堆序，就向上交换

这叫“向上调整”。

#### `heapify`
当堆顶元素变小了，或者堆顶和末尾交换后，根节点可能不再满足堆序。

做法：

- 和左右孩子中更合适的那个比较
- 如果需要，就向下交换

这叫“向下调整”。

## 典型例子
### 大根堆示例
数组：

```text
[9, 7, 8, 3, 5, 6]
```

对应的完全二叉树满足：

- `9 >= 7, 8`
- `7 >= 3, 5`
- `8 >= 6`

所以它是一个大根堆，堆顶 `9` 就是最大值。

### 小根堆示例
数组：

```text
[2, 4, 3, 8, 7, 5]
```

满足：

- `2 <= 4, 3`
- `4 <= 8, 7`
- `3 <= 5`

所以它是一个小根堆，堆顶 `2` 就是最小值。

## 和优先级队列的关系
很多语言中的优先级队列，本质都是堆。

例如 Java 的：

```java
PriorityQueue<Integer>
```

默认就是小根堆。

也就是说：

- 插入元素后自动维护堆序
- 每次弹出堆顶，就是当前最小值

## 复杂度
- 访问堆顶：`O(1)`
- 插入一个元素：`O(log N)`
- 调整堆序：`O(log N)`
- 建堆方式不同，总代价会不同

这一题先建立概念，后面几题会继续展开：

- `heapify`
- `pop` 后如何重建堆
- 为什么从底向上建堆是 `O(N)`

## 易错点
- 堆不是二叉搜索树，左右子树之间没有“左小右大”关系。
- 完全二叉树不等于满二叉树。
- 大根堆/小根堆只要求父子关系，不要求整层有序。
- 堆顶是全局最值，但其他位置并不是完整有序的。
